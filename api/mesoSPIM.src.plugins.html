property metadata_file_info: str¶

Return the file name for the current metadata file. This function should be updated as needed and is called after self.open() for each tile Default appends ‘_meta.txt’ to the filename (i.e. WriteRequest.uri) Appends to attrs to be used for writing metadata

self.metadata_file # Actual file where metadata is stored self.metadata_file_describes_this_path # The specific file described by self.metadata_file

Reasonable defaults are set for ImageWriter that are 1_Tile=1_file This may need to be overwritten if FileNaming(SingleFileFormat=True)
