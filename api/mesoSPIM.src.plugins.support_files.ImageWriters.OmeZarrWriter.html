class mesoSPIM.src.plugins.support_files.ImageWriters.OmeZarrWriter.omezarr_writer.ChunkScheme(base: Tuple[int, int, int] = (1, 1024, 1024), target: Tuple[int, int, int] = (64, 64, 64))[source]¶

Bases: object

Per-level chunking policy that evolves from base -> target as levels increase. - base applies at level 0 (e.g., z=8, y=512, x=512) - at each level l, we:

zc = min(target.z, base.z * 2**l) # z chunk grows toward target yc = max(target.y, max(1, base.y // 2**l))# y/x chunks shrink toward target xc = max(target.x, max(1, base.x // 2**l))

Then we clamp to the level’s array shape.
