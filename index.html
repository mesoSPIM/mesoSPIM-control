mesoSPIM Control¶

mesoSPIM-control is the open-source Python/PyQt acquisition software for mesoSPIM light-sheet microscopes. It drives all hardware, manages acquisition protocols, and provides a GUI for multichannel, multiview, and tiled imaging of large cleared-tissue samples.

Tip

New to mesoSPIM? Start with the Getting Started page to install the software and run it in demo mode within minutes.
