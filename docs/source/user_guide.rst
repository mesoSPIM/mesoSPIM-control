Typical workflows
-----------------

.. rubric:: Basic single-channel acquisition

1. Start mesoSPIM-control (demo mode or real hardware).
2. Select or create a configuration file.
3. Set camera exposure, laser power, and scanning parameters.
4. Start acquisition and monitor progress.
